# 🎤 AI Interview Coach

An AI-powered Interview Coach built using **n8n**, **Groq LLM**, and **Simple Memory**. It conducts mock technical interviews, evaluates responses, and provides AI-powered feedback to help users improve their interview skills.

## 🚀 Features

- 🎤 Interactive Mock Interviews
- 🤖 AI-Powered Question Generation
- 📝 Answer Evaluation
- ⭐ Score for Every Response
- 💡 Constructive Feedback
- 🧠 Conversational Memory
- ⚡ Workflow Automation with n8n

## 🛠️ Tech Stack

- n8n
- Groq LLM
- Simple Memory

## 📦 Installation

1. Download the `ai-interview-coach.json` workflow.
2. Import the workflow into n8n.
3. Configure your Groq Chat Model credentials.
4. Save and activate the workflow.
5. Open the chat and start your mock interview.
